User Stories from Product Backlog that we will address in this sprint:
a) As a user, I can remove a directory from the file system including all sub-directories of it.		

b) As a user, I can list the contents of one or more directories recursively.					
c) As a user, I can move a file or a directory from one path to another path.				
d) As a user, I can copy a file or a directory to new path.							
e) As a user, I can search files and directories in the file system.									
f) As a user, I can display the entire file system as a tree structure.
