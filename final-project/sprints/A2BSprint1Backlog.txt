A2B Sprint 1 Backlog.  
November 16 to November 22


User Stories from Product Backlog that we will address in this sprint:
a) As a user, I can remove a directory from the file system including all sub-directories of it.
b) As a user, I can create multiple directories at once.
c) As a user, I can list the contents of one or more directories recursively.
d) As a user, I can display the contents of one or more files on the console.
e) As a user, I can move a file or a directory from one path to another path.
f) As a user, I can copy a file or a directory to new path.

d-1. (cat) Write extra code to account for diplaying contents of multiple files
d-2. (cat) Test cat class to make sure it works with all possible inputs

e-1. (mv) Create a mv class and come up with solution code to move an item from one path to another.
f-1. (cp) Create a cp class and implement adjusted solution code to copy an item to new path.
g-1. (man) update new commands for man command
